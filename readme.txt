Show automatically the followers you have on each social network. 

== Description ==

This plugin allows you to show automatically the number of followers, friends and fans you have on each social network together with the corresponding social profile.
You just need register in www.showmb.com and use the user ID given by the platform.


== Installation ==

How do I get my user ID?

1. Go to showmb.com
1. Use your Facebook's account to register.
1. Access to your dashboard to get your user ID.

How do I install and activate the plugin? 

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Go to Appearance and select the Widgets option. Now select the ShowMB widget and drag it to the desired location. Write your ShowMB user ID and save it. 

== Frequently Asked Questions ==

= Whats happens if I write my user ID wrong? =

Nothing.


= Can I have more than one ShowMB widget running at the same time? =

Yes, if your blog has different authors you can include more than one widget (one for each author).


= How often is the number of followers updated? =

Your number of followers are updated periodically, however if you want an instant update, you can go to ShowMB and update your follower numbers from there. As soon as you do it they will automatically renew in your blog.
